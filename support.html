AzAd Browser Kit

Visit the browser extension store for your browser (Chrome Web Store, Firefox Add-ons, etc.), search for "AzAd Browser Kit", and click "Add to [Browser]". The extension will install automatically and appear in your extension menu.

AzAd Browser Kit officially supports Chrome, Edge, Firefox, and Opera. Safari support is in development. Check the extension page for your browser to see the latest compatibility information.

Try: 1) Refresh the page, 2) Disable and re-enable the extension, 3) Clear extension data and cookies, 4) Check if the site allows extensions, 5) Restart your browser. If issues persist, report on GitHub Issues with your browser version and error details.

AzAd Smart Downloader

Visit the Chrome Web Store, search for "AzAd Smart Downloader", and click "Add to Chrome". The extension will install automatically. Pin it to your toolbar for easy access by clicking the puzzle icon and selecting the pin icon next to ASD.

AzAd Smart Downloader scans for and supports all common file types: images (PNG, JPG, GIF, WebP, SVG), documents (PDF, DOC, XLSX), audio (MP3, WAV, AAC), video (MP4, WebM, MKV), and archives (ZIP, RAR). It detects files from images, links, media tags, and inline styles.

Click the ASD icon in your toolbar, then click "Scan This Page" in the popup. The extension scans the current webpage for all downloadable files (from links, images, media tags, etc.). Results open in a new tab showing thumbnails or emoji previews, organized by file type and source.

Files are automatically organized in folders by hostname (website source) and file type (images, documents, audio, video, archives). This creates a clean structure in your Downloads folder. For faster downloads, go to Chrome Settings → Downloads and uncheck "Ask where to save each file before downloading".

Yes! AzAd Smart Downloader is privacy-first. All scan results and settings remain local to your browser. We don't collect, track, or send data about your scans to external servers. Read our Privacy Policy for complete details.

ASO-Suite Pro

ASO-Suite Pro is a professional WordPress admin security and optimization toolkit with 22+ modular tools. It provides advanced control and automation for WordPress security scanning, vulnerability detection, hardening recommendations, access logging, and performance optimization. Visit asosuitepro.com for more details.

ASO-Suite Pro requires WordPress 5.0+. For best results, use the latest WordPress version with all security updates. It's compatible with most modern WordPress themes and hosting configurations. Check the documentation for specific version requirements.

Install ASO-Suite Pro from your WordPress dashboard: Go to Plugins → Add New → Search "ASO-Suite Pro" → Click Install → Activate. Or download from GitHub and upload manually. After activation, the admin dashboard will have ASO-Suite Pro settings and tools.

No. ASO-Suite Pro is optimized for minimal performance impact. It uses efficient background processes and caching to avoid slowing your site. Security scans and optimizations are scheduled to run during off-peak hours. Most users report no noticeable performance degradation.
